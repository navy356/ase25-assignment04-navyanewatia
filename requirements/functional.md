# Functional Requirements
- As a customer, I want to browse meals offered by the catering service so I can see/select from available options.
- As a customer, I want to search meals by name, ingredient or category (e.g., vegan, quick meals, gluten-free) so I can easily find what I'm looking for.
- As a customer, I want to view the list of ingredients required for each meal so I can see what will be ordered.
- As a customer, I want to order ingredients for a selected meal so I can prepare the meal.
