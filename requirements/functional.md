# Functional Requirements
- As a customer, I want to browse meals offered by the catering service so I can view and select from available options.
- As a customer, I want to search meals by name, ingredient or category (e.g., vegan, quick meals, gluten-free) so I can easily find meals that meet my preferences.
- As a customer, I want to view the list of ingredients required for each meal so I can see what ingredients will be ordered.
- As a customer, I want to order ingredients for a selected meal so I can prepare that meal.
- As a customer, I want to schedule ingredient deliveries (same-day or next-day) so that they arrive as per my needs and availibility.
